# DJ Learning Journey — Milestones

---

## 2026-03-30 · Party Mix 阶段
- 第一次系统研究 Party Mix 控制器布局，逐一梳理所有按键功能
- 排查了 PFL/Cue 耳机音量过小的问题，根本原因是 EPOS Adapt 360（蓝牙办公耳机）不适合直插控制器——需改用有线 TRS 耳机
- 理解了 Gain vs Channel Fader 在信号链中的不同层级与作用
- 了解了 Sampler 模式（Performance Pads 切换到 Sampler 模式后可存储并触发音频片段）

## 2026-04-11 · DJ_Pool 搭建 + 户外需求萌生
- yt-dlp + ffmpeg 安装完成（Windows PowerShell，winget 方式，验证版本号通过）
- DJ_Pool 文件夹结构设计并创建，按风格 + BPM 区间命名：`House_bpm 120-128`、`DeepHouse_bpm 118-124`、`TechHouse_bpm 126-132`、`Techno_bpm 130-145`、`Trance_bpm 128-140`、`HipHop_bpm 85-100`、`RnB_bpm 60-90`、`Pop_bpm 100-128`、`Unsorted`、`Misc`
- 确认 Serato Library 手动添加文件夹的方式（Files 面板拖入）
- 首次明确提出户外演出需求：DJ_Pool 文件夹能否迁移到 iPad 或 MacBook 出去演出？——iPad 方案当日排除（Serato 不支持 iOS）

## 2026-04-12 · Beat Grid 理解
- 开始系统整理 DJ 学习资源（YouTube 教程 + 付费课程）
- 建立学习路径框架（Serato Lite 功能限制、Party Mix 硬件特性）
- 深入理解 Beat Grid 概念：Sync 按钮依赖 Beat Grid 分析结果，而非实时"听"音乐对拍
- 学习了在 Serato 中手动修复错误 Beat Grid 的操作流程

## 2026-04-16 · 设备局限确认
- 深入对比 Party Mix 与升级机型的功能差距
- 确认 Filter 缺失是 Party Mix 最实质的短板（影响远大于没有 Echo）——Party Mix 够练基础，但不适合正式演出场合
- 理解了歌曲结构（Intro / Breakdown / Build / Drop / Outro）与 DJ 混音窗口的对应逻辑
- 了解了 Echo / Reverb / Flanger 的实际用途与使用场景

## 2026-04-17 · 升级主控制器
